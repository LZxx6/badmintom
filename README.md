# 🏸 1314 林丹杯积分排行系统 (云同步版)

这是一个轻量级、纯前端实现且支持云端数据同步的羽毛球赛事积分与排名管理系统。专为球友日常约球、内部比赛打卡设计，支持单双打积分计算、赛季管理以及选手能力值可视化。

## ✨ 核心特性

* **☁️ 云端同步**：基于 [JSONBin.io](https://jsonbin.io/) 实现轻量级无后端数据存储，多设备实时同步比赛记录与排行榜。
* **🏅 智能排行榜**：
    * 自动分离 **单打 (Singles)** 与 **双打 (Doubles)** 排行榜。
    * 自动计算积分与战绩（胜、平、负）。默认规则：胜一场 +1 分，负一场 -1 分，平局不加减分。
    * 支持同分情况下的胜场优先排序机制。
* **📅 赛事日历**：直观展示当月有比赛记录的日期，方便回顾赛程。
* **👤 选手数据卡片**：
    * 内置选手轮播图，快速预览核心战绩。
    * 集成 **Chart.js** 绘制六维能力雷达图（进攻、防守、速度、体能、技巧、心态）。
    * 支持在本地自定义调节每位选手的能力值。
* **📁 赛季管理**：支持按赛季（如 2026 赛季、2027 赛季）归档和切换比赛数据，互不干扰。
* **🌓 响应式与暗黑模式**：完美适配手机端与 PC 端，内置一键切换的深夜模式 (Dark Mode)。

## 🛠️ 技术栈

* **前端**：HTML5, CSS3 (原生变量与 Flex/Grid 布局), Vanilla JavaScript (原生 JS)
* **图表**：[Chart.js](https://www.chartjs.org/) (CDN 引入)
* **后端/数据库**：[JSONBin.io](https://jsonbin.io/) (REST API)

## 🚀 快速开始

本项目无需复杂的 Node.js 环境或数据库部署，只需一个浏览器即可运行。

### 1. 本地运行
1. 将代码保存为 `index.html`。
2. 双击在浏览器中打开，或者使用 VS Code 的 `Live Server` 插件运行。

### 2. 配置云端同步 (重要)
为了让数据能够在你的不同设备间同步，你需要配置自己的 JSONBin 密钥：

1. 前往 [JSONBin.io](https://jsonbin.io/) 注册一个免费账号。
2. 创建一个空的 JSON Bin，并获取它的 **Bin ID**。
3. 在你的账户设置中生成一个 **API Key**（或者使用 Master Key）。
4. 打开 `index.html`，找到代码行约 `313` 行的配置区，将你的凭证填入：
